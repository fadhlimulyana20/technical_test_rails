# README

Technical Test Rails Application

## Route
* / (index)
* /book (book-list)
* /book/new (create book)
* /book/api/v1/book (Book List API)
* /author/api/v1/author (Author List API)

## Install
```sh
bundle
rails db:migrate
rails db:seed
```

## Test
```sh
rails test
```

## Run
```sh
rails s
```
